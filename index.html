1. Child wears the button

The device is attached to clothing like a normal button — discreet and hard to notice.

2. Data streaming

Button sends GPS + movement + SOS events to our secure backend (encrypted).

3. AI monitoring

Anomaly detection runs in our AI service and flags risk.

4. Parent alert

Parent receives instant alerts with location + actions.

Hardware Prototype (MVP)

The first version is built using Arduino + GPS + GSM module (SIM800L). It sends real-time coordinates via SMS/GPRS. This MVP proves the core functionality before miniaturization.

How Our AI Works

Anomaly Detection

Our AI flags unusual movement, speed, route deviation, geofence exit, and tamper attempts.

Real‑time Alerts

If unusual events are detected, parents immediately receive alerts via app/SMS.

Low‑latency Microservice

Lightweight Python AI microservice analyzing device signals in real time.
