Our Solution — SmartButton.AI

What it is

SmartButton.AI is a mini, hidden, button-sized wearable that attaches to clothing and connects to a secure parent app.

Core features

  • Live GPS tracking
  • One-tap SOS alert
  • AI-powered anomaly detection
  • Safe-zone (geo-fence) warnings
  • Tamper detection
Roadmap & MVP →

How it works (simple)

1. Child wears the button

The device is attached to clothing like a normal button — discreet and hard to notice.

2. Data streaming

Button sends GPS + movement + SOS events to our secure backend (encrypted).

3. AI monitoring

Anomaly detection runs in our AI service and flags risk.

4. Parent alert

Parent receives instant alerts with location + actions.

Hardware Prototype (MVP)

The first version is built using Arduino + GPS + GSM module (SIM800L). It sends real-time coordinates via SMS/GPRS. This MVP proves the core functionality before miniaturization.

How Our AI Works

Anomaly Detection

Our AI flags unusual movement, speed, route deviation, geofence exit, and tamper attempts.

Real-time Alerts

If unusual events are detected, parents immediately receive alerts via app/SMS.

Low-latency Microservice

Lightweight Python AI microservice analyzing device signals in real time.

Why SmartButton.AI is unique

Hidden & Discreet

Designed to look like a normal clothing button — not removable or obvious to attackers.

AI-first Detection

Not just GPS — AI analyzes motion, routes, and tamper signals to reduce false alarms and catch real danger early.

Low Power & Affordable

Optimized for long battery life and low cost so it’s accessible for families at scale.
