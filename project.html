What We Will Build (MVP)

1) Device Simulation

Python script that emits GPS, battery level, movement events, and SOS triggers to our API in real-time.

2) Backend & Database

Node.js + Express API for location, SOS, authentication, and history. Data stored in Postgres/MySQL.

3) AI Microservice

Python-based anomaly detection: geofence exit, speed anomaly, tamper detection. Uses heuristic + light ML.

4) Parent Dashboard

React dashboard with live map, SOS feed, child timeline, and safe‑zone management.

5) Hardware MVP (Arduino Prototype)

Early physical device using:
• Arduino board
• GPS module (NEO‑6M)
• GSM module (SIM800L)

Sends real-time coordinates via SMS/GPRS and supports SOS events. Validates the core tracking & emergency alert system before custom PCB development.

System Architecture

End‑to‑end pipeline: Device → Backend → AI Engine → Parent Dashboard.
